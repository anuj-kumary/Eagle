Badge

Badge can be used to show the status information of the user or some count information such as notification numbers or number of items in the cart.

Profile Badge

Badge on Avatar can show different user status, such as offline and online. We have different badges that can be used with avatar. It can be used with 3 variants : status--online | status--idle | status--offline
