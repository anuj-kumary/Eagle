Button

The Button component is used to trigger an action or event. EagleUI provides various types of buttons that you can use in your website's such as submitting a form, opening a dialog, canceling an action, or performing a delete operation.

Solid Buttons
EagleUI provides buttons with different state such as Success, Error, Warning, Dark and Light. 
            
<button class="btn btn__default">Primary</button>
<button class="btn btn--secondary">Secondary</button>
<button class="btn btn__success">Success</button>
<button class="btn btn__error">Error</button>
<button class="btn btn__warn">Warning</button>
<button class="btn btn__dark">Dark</button>
<button class="btn btn__light">Light</button>
Link Buttons
Want to attach a link action, at a button tap? Use btn__link Button
See More 
            
<button class="btn">
<a class="btn__link" href="#">Link</a>
</button>
<a class="btn__link" href="#">See More</a>
Icon Buttons
Add the btn class to the button element and a fontawesome icon. 
            
<button class="btn"><i class="icon fa fa-home"></i> Home</button>
<button class="btn"><i class="icon fa fa-bars"></i> Menu</button>
<button class="btn"><i class="icon fa fa-trash"></i> Trash</button>
<button class="btn"><i class="icon fas fa-times"></i> Close</button>
<button class="btn"><i class="icon fa fa-folder"></i> Folder</button>
Outlined Buttons
In need of a button, but not the hefty background colors they bring? Replace the default modifier classes with the btn--outlined and outlined__success ones to remove all background images and colors on any button.
Floating Buttons
            
<button class="btn__floating"><i class="floating__icon fas fa-chevron-up"></i></button>
