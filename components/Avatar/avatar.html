Avatar
The Avatar component is used to represent users. And should only be used for users.
Profile Avatar

Profile Avatars can be created using the class 'avatar__img', inside the img tag.

              
        <img class="avatar__img" src="https://rb.gy/mvc9gm" alt="Avatar" />
        <img class="avatar__img" src="https://rb.gy/xqmshi" alt="Avatar" />
        <img class="avatar__img" src="https://rb.gy/2trpns" alt="Avatar" />
Size Avatar

Avatar is available in 3 different sizes. You can use image in Avatar. You just need to include class Size of the avatars can be changed by adding sm | md | lg class to the existing avatar class.

              
        <img class="avatar__img lg" src="https://rb.gy/mvc9gm" alt="Avatar" />
        <img class="avatar__img md" src="https://rb.gy/xqmshi" alt="Avatar" />
        <img class="avatar__img sm" src="https://rb.gy/2trpns" alt="Avatar" />
